# 功能規格清單

以 Heptabase 的功能概念為藍本，拆成 10 個模組。每個功能標註優先級：

- **P0** = 核心，沒有就不是這個產品（MVP 必做）
- **P1** = 重要，完整體驗必備（第二階段）
- **P2** = 加分，進階/生態系功能（第三階段）

---

## 1. 卡片系統（Card）

卡片是最小知識單位，所有內容都以卡片形式存在，可被多處引用。

| 優先級 | 功能 | 說明 |
| --- | --- | --- |
| P0 | 卡片 CRUD | 建立、編輯、封存、刪除卡片；每張卡片有標題與內文 |
| P0 | 卡片庫（Card Library） | 集中列表檢視所有卡片，支援排序（建立/更新時間、標題） |
| P0 | 一卡多處引用 | 同一張卡片可出現在多個白板，內容單一來源、處處同步 |
| P1 | 卡片資訊面板 | 顯示建立/更新時間、所屬白板、反向連結、標籤 |
| P1 | 卡片合併與拆分 | 把選取的文字拆成新卡片（turn into card）；卡片合併 |
| P1 | 封存與垃圾桶 | 封存不常用卡片；刪除進垃圾桶，可還原或永久刪除 |
| P2 | 卡片版本歷史 | 檢視與還原歷史版本 |
| P2 | 卡片模板 | 以模板快速建立固定結構的卡片 |

## 2. 編輯器（Editor)

Block-based 富文字編輯器，是卡片、日誌共用的核心元件。

| 優先級 | 功能 | 說明 |
| --- | --- | --- |
| P0 | 基本區塊 | 段落、H1–H3 標題、無序/有序清單、待辦（checkbox）、引言、分隔線 |
| P0 | 行內格式 | 粗體、斜體、底線、刪除線、行內程式碼、螢光標記、連結 |
| P0 | `/` 指令選單 | 斜線呼叫區塊選單快速插入 |
| P0 | Markdown 快捷輸入 | `#`、`-`、`[]`、`>` 等自動轉換為對應區塊 |
| P0 | 雙向連結語法 | `[[` 觸發卡片搜尋，插入卡片連結；連到不存在的卡片時自動建立 |
| P1 | 巢狀清單與折疊 | 清單縮排層級、toggle 折疊區塊 |
| P1 | 圖片區塊 | 貼上/拖曳上傳圖片，支援縮放 |
| P1 | 程式碼區塊 | 語法高亮、語言選擇 |
| P1 | 表格區塊 | 簡單表格編輯 |
| P1 | 區塊拖曳排序 | 拖曳 handle 重新排列區塊 |
| P1 | 卡片嵌入（embed） | 在內文中嵌入整張卡片並可展開編輯 |
| P2 | 數學公式 | LaTeX 行內與區塊公式 |
| P2 | 影音與檔案區塊 | 影片、音訊、附件上傳 |
| P2 | 網頁書籤區塊 | 貼上 URL 產生預覽卡 |

## 3. 白板（Whiteboard）

無限畫布，用空間排列與連線來思考。

| 優先級 | 功能 | 說明 |
| --- | --- | --- |
| P0 | 無限畫布 | 平移、縮放（滑鼠/觸控板手勢）、fit-to-view |
| P0 | 卡片上板 | 在白板上新建卡片；從卡片庫拖入既有卡片 |
| P0 | 移動與縮放卡片 | 拖曳移動、調整卡片顯示大小、多選（框選/shift 點選） |
| P0 | 連線（Edge） | 卡片間拉線，支援箭頭方向、線上文字標籤 |
| P0 | 白板 CRUD | 建立、重新命名、刪除白板；白板列表 |
| P1 | Section 區域 | 在白板上圈出具名區域，卡片可歸入；區域可整體移動與折疊 |
| P1 | 文字與便利貼元素 | 白板上直接放輕量文字/便利貼（不入卡片庫） |
| P1 | 卡片顏色 | 卡片在白板上的顏色標記 |
| P1 | 子白板 / 白板巢狀 | 白板中嵌入另一個白板的入口卡 |
| P1 | 對齊與自動排列 | 對齊輔助線、選取後自動排列/等距分佈 |
| P1 | 小地圖（minimap） | 畫布縮圖導航 |
| P2 | 心智圖模式 | 樹狀節點快速展開的 mind-map 佈局 |
| P2 | 手繪與圖形 | 筆刷、箭頭、幾何圖形 |
| P2 | 白板匯出 | 匯出為圖片 / PDF |

## 4. 雙向連結與知識網絡

| 優先級 | 功能 | 說明 |
| --- | --- | --- |
| P0 | 反向連結面板 | 每張卡片顯示「哪些卡片連到我」，含上下文預覽 |
| P1 | 未連結提及 | 偵測內文出現卡片標題但未加連結的段落，一鍵轉為連結 |
| P1 | 連結預覽 | hover 卡片連結顯示內容預覽浮窗 |
| P2 | 全域知識圖譜 | 卡片關係的 graph view 視覺化 |

## 5. 標籤與資料庫（Tag）

| 優先級 | 功能 | 說明 |
| --- | --- | --- |
| P0 | 標籤 CRUD | 為卡片加/移除標籤；標籤列表頁 |
| P1 | 標籤即資料庫 | 每個標籤有自己的頁面，以表格列出所有掛此標籤的卡片 |
| P1 | 自訂屬性（Properties） | 標籤下可定義欄位：文字、數字、單選、多選、日期、checkbox |
| P1 | 表格檢視 | 依屬性排序、篩選 |
| P1 | 看板檢視（Kanban） | 依單選屬性分欄，拖曳改值 |
| P2 | 巢狀標籤 | `專案/子專案` 階層標籤 |
| P2 | 媒體牆檢視 | 以卡片縮圖呈現的 gallery view |

## 6. 日誌（Journal）

| 優先級 | 功能 | 說明 |
| --- | --- | --- |
| P0 | 每日一頁 | 依日期自動建立日誌頁，時間軸無限捲動 |
| P0 | 快速捕捉 | 開 App 直達今日日誌，隨手記錄 |
| P1 | 日曆導航 | 月曆跳轉任意日期 |
| P1 | 日誌內容上板 | 把日誌中的段落拆成卡片、拖到白板 |
| P2 | 回顧（On this day） | 顯示歷史上的今天寫過什麼 |

## 7. 搜尋與導航

| 優先級 | 功能 | 說明 |
| --- | --- | --- |
| P0 | 全文搜尋 | 搜尋卡片標題與內文，結果高亮 |
| P0 | 快速開啟（Cmd+K / Cmd+O） | 命令面板：搜尋跳轉卡片/白板、執行指令 |
| P1 | 進階篩選 | 依標籤、白板、日期範圍過濾搜尋結果 |
| P1 | 側邊欄 | 白板樹、卡片庫、標籤、日誌、垃圾桶的固定導航 |
| P1 | 分割視窗 | 左右並排開兩張卡片/白板 |
| P1 | 多分頁 | 像瀏覽器一樣開多個 tab；歷史前進/後退 |
| P2 | 最近開啟 | 最近瀏覽的卡片與白板清單 |

## 8. 閱讀與標註（PDF / Highlights）

| 優先級 | 功能 | 說明 |
| --- | --- | --- |
| P1 | PDF 匯入與閱讀 | 上傳 PDF，內建閱讀器 |
| P1 | PDF 劃線標註 | 選取文字劃線、加註解；標註自動成為可引用的 highlight |
| P1 | 標註上板 | 把 highlight 拖到白板成為卡片，保留跳回原文位置的連結 |
| P2 | 圖片區域標註 | 對 PDF 截圖區域做標註 |
| P2 | EPUB / 網頁文章 | 更多閱讀格式支援 |
| P2 | Readwise 同步 | 匯入 Readwise 的 highlights |

## 9. 同步、匯入匯出與平台

| 優先級 | 功能 | 說明 |
| --- | --- | --- |
| P0 | 本地持久化 | 資料離線可用，本地優先儲存 |
| P0 | Markdown 匯出 | 單卡與整庫匯出為 .md |
| P1 | 雲端同步 | 帳號系統 + 多裝置即時同步（含離線衝突處理） |
| P1 | Markdown / Notion / Evernote 匯入 | 批量匯入既有筆記 |
| P1 | 桌面 App | macOS / Windows（Tauri 或 Electron 打包） |
| P2 | 行動 App | iOS / Android（至少支援日誌捕捉與卡片閱讀） |
| P2 | 端到端加密選項 | 進階隱私保護 |

## 10. 協作與分享

| 優先級 | 功能 | 說明 |
| --- | --- | --- |
| P2 | 白板公開分享 | 產生唯讀分享連結 |
| P2 | 即時協作 | 多人同時編輯白板與卡片（presence、游標） |
| P2 | 評論 | 對卡片/區塊留言討論 |
| P2 | 工作區與權限 | 多工作區、成員邀請、角色權限 |
